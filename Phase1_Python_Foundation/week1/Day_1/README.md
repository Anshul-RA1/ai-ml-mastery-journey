# Day 1 — Variables, Data Types, Strings & Operators

## What I Learned

- Python basics: variables, data types (str, int, float, bool)
- String methods: .split(), .strip(), .upper(), .replace(), f-strings
- Arithmetic, comparison, and logical operators
- User input with input() and type conversion

| 1   | Bio Generator         | Easy       | Variables, f-strings, type()          |
| 2   | Salary Calculator     | Easy       | input(), float conversion, formatting |
| 3   | BMI Calculator        | Medium     | Math operators, if/elif/else          |

## Key Takeaways

- f-strings are the cleanest way to format output in Python
- input() always returns a string — convert with int() or float()
- snake_case is the Python naming convention for variables
